About Employee Management System

This Employee Management System is a web application built using Django, a high-level Python web framework. It allows organizations to manage their employee data efficiently, including information such as first name, last name, ID, gender, hiring date, department, manager, and salary.

Features:

  • User-friendly interface for managing employee data.
  • Secure authentication system for administrators.
  • Integration with Bootstrap for a modern and responsive design.
  • Support for adding, editing, and deleting employee records.
  • Ability to view employee data in a table format.

Technologies Used:

  • Django: Python web framework
  • Bootstrap: Front-end framework
  • SQLite: Relational database management system

For more information or to contribute to the project, visit the GitHub repository: Employee Management System
